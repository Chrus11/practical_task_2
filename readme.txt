1.Для анализа был скачан датасет shopping_trends.csv с ресурса kaggle.com (Набор данных о покупательских предпочтениях клиентов).
Для загрузки данных и визуализации были импортированы библиотеки pandas и matplotlib.
2. С помощью describe и info проведен анализ данных массива: типы данных, наличие пропущенных значений и основные статистические характеристики.
Данные чистые, пропущенных значений нет. В преобразованиях и масштабировании не нуждаются.
3. По двум столбцам ("Gender" и "Category") был произведен подсчет количества заказов, совершенных мужчинами и женщинами и количества заказов по отдельным категориям.
4. На основе посчитанных данных были построены две круговые диаграммы: "Распределения заказов в зависимости от пола покупателей" и Распределение заказов по категории товаров".
